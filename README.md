# Concrete Knowhow

Free concrete, gravel, mulch, topsoil, sand, and sod calculators.

Built with Next.js 14 App Router, TypeScript, and Tailwind CSS — 100% static export.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deployed to Vercel at [concrete.mrknowitall.net](https://concrete.mrknowitall.net).
